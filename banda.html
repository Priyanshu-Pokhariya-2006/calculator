About Our Project

This is a simple scientific calculator built with HTML, CSS, and JavaScript (implied by the functionality, though not explicitly shown in code snippets). It helps you perform basic math operations, trigonometry functions, unit conversions, and more. Our goal was to create a clean, responsive, and educational tool.

Frontend: HTML, CSS, JavaScript

Framework/Library: Bootstrap 5.3

We designed, structured, and styled the calculator with responsive sections and a clean UI, leveraging Bootstrap for layout and component styling.
