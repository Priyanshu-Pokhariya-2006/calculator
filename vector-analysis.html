Vector Analysis

Definition

Vector analysis deals with quantities that have both **magnitude** and **direction**.

Vectors are used to describe **forces**, **velocities**, **displacement**, and many physical quantities.

Understanding Vectors

A vector is represented as an arrow or as an ordered pair or triple.

  • **Magnitude**: length of vector
  • **Direction**: the angle or orientation
  • **Components**: (x, y) or (x, y, z)

Key Properties

  • **Addition**: a + b = (a₁+b₁, a₂+b₂)
  • **Scalar multiplication**: k·a = (ka₁, ka₂)
  • **Magnitude**: |a| = √(a₁² + a₂²)
  • **Unit vector**: a / |a|

Identities 

  • **Dot product**: a·b = |a||b|cosθ
  • **Cross product**: |a×b| = |a||b|sinθ
  • a·a = |a|²
  • **Distributive**: a·(b + c) = a·b + a·c

Examples

1. Magnitude of (3,4) → √(3²+4²)=5

2. Dot product: (2,3)·(4,1)= 8+3=11

3. Unit vector of (6,8) → (6/10, 8/10)
