Complex Numbers

Definition

A **complex number** is a number that can be written in the **standard form** $z = a + bi$, where $a$ and $b$ are real numbers and $i$ is the **imaginary unit** with $i^2 = -1$.

$a$ is the **real part** ($\text{Re}(z)$) and $b$ is the **imaginary part** ($\text{Im}(z)$).

Understanding Complex Numbers

Complex numbers extend real numbers, allowing solutions for equations like $x^2 + 1 = 0$.

  • They are plotted on the **Complex Plane** (or **Argand Diagram**).
  • The horizontal axis represents the **real part** ($a$).
  • The vertical axis represents the **imaginary part** ($b$).
  • Every point $(a, b)$ on the plane corresponds to a complex number $z = a + bi$.

Key Properties

  • **Powers of i:** $i^2 = -1$, $i^3 = -i$, $i^4 = 1$. The pattern repeats every four powers.
  • **Real numbers:** Are complex numbers where the imaginary part is zero ($b = 0 \rightarrow z = a$).
  • **Pure imaginary numbers:** Are complex numbers where the real part is zero ($a = 0 \rightarrow z = bi$).
  • **Magnitude/Modulus:** $|z| = \sqrt{a^2 + b^2}$ (the distance from the origin to $z$).
  • **Conjugate:** $\bar{z} = a - bi$ (the sign of the imaginary part is flipped).

Identities

  • **Multiplication with Conjugate:** $z \cdot \bar{z} = |z|^2 = a^2 + b^2$
  • **Addition:** $(a + bi) + (c + di) = (a + c) + (b + d)i$
  • **Multiplication:** $(a + bi)(c + di) = (ac - bd) + (ad + bc)i$
  • **Division:** $\frac{a + bi}{c + di} = \frac{(a + bi)(c - di)}{c^2 + d^2}$ (Multiply by conjugate of denominator)
  • **Polar form:** $z = r(\cos \theta + i \sin \theta)$, where $r=|z|$ and $\theta=\arg(z)$
  • **Euler form:** $z = r e^{i\theta}$ (This is Euler's formula: $e^{i\theta} = \cos \theta + i \sin \theta$)

Examples

1. Addition:
$(3 + 4i) + (2 - 5i) = (3 + 2) + (4 - 5)i = 5 - i$

2. Multiplication:
$(1 + 2i)(3 + 4i) = 3 + 4i + 6i + 8i^2 = 3 + 10i - 8 = -5 + 10i$

3. Magnitude:
If $z = 3 + 4i$, then $|z| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$

4. Conjugate:
If $z = 5 - 7i$, then $\bar{z} = 5 + 7i$

5. Polar form:
If $z = 1 + i$, then $r = \sqrt{1^2 + 1^2} = \sqrt{2}$, and $\theta = \arctan(1/1) = 45^\circ$. Thus $z = \sqrt{2}(\cos 45^\circ + i \sin 45^\circ)$
