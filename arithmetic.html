Arithmetic

Definition

**Arithmetic** is the branch of mathematics that deals with basic number operations such as **addition**, **subtraction**, **multiplication**, and **division**.

It forms the foundation of all advanced mathematics and daily quantitative reasoning.

Understanding Arithmetic

Arithmetic helps perform everyday computations and understand numerical relationships. The four main operations are:

  • **Addition** (+) combines quantities (e.g., $2 + 3 = 5$)
  • **Subtraction** (-) finds the difference between quantities (e.g., $5 - 2 = 3$)
  • **Multiplication** ($\times$) repeats addition (e.g., $2 \times 3 = 6$)
  • **Division** ($\div$) splits quantities into equal parts (e.g., $6 \div 3 = 2$)

Key Properties

  • **Commutative Property:** $a + b = b + a$ (for addition and multiplication)
  • **Associative Property:** $(a + b) + c = a + (b + c)$ (for addition and multiplication)
  • **Distributive Property:** $a(b + c) = ab + ac$
  • **Identity Elements:** $0$ for addition ($a + 0 = a$), $1$ for multiplication ($a \times 1 = a$)
  • **Inverse:** additive inverse ($-a$, where $a + (-a) = 0$), multiplicative inverse ($1/a$, where $a \times (1/a) = 1$)

Identities

  • Difference of Squares: $(a - b)(a + b) = a^2 - b^2$
  • Perfect Square (Addition): $(a + b)^2 = a^2 + 2ab + b^2$
  • Perfect Square (Subtraction): $(a - b)^2 = a^2 - 2ab + b^2$
  • Percentage formula: $\text{Percent} = (\text{part}/\text{whole}) \times 100$

Examples

1. Addition: $45 + 27 = 72$

2. Division: $120 \div 6 = 20$

3. Identity: $(a + b)^2$ example: $(3 + 4)^2 = 7^2 = 49$. Also $3^2 + 2(3)(4) + 4^2 = 9 + 24 + 16 = 49$
